## рџЊџ Overview

**VEDYRA Labs** is a modern startup website showcasing the vision of a technology company focused on transforming the automotive ecosystem through intelligent vehicle health analytics.

The project has been engineered as a **production-quality React application** using modern frontend technologies including **TanStack Start**, **React 19**, **Tailwind CSS v4**, **Nitro Server-Side Rendering**, and **Vite**. Rather than being a simple landing page, it demonstrates how scalable frontend architecture, clean UI engineering, responsive design, and modern development workflows can be combined into a polished production-ready web experience.

The repository serves as both the official website for VEDYRA Labs and a showcase of modern frontend engineering practices.

## Clone the Repository

```bash

```

Move into the project directory:


---

## Install Dependencies

Using Bun:

```bash
bun install
```

or using npm:

```bash
npm install
```

---

## Development Server

Start the local development server:

```bash
bun run dev
```

The application will be available locally during development with Hot Module Replacement (HMR) enabled.
